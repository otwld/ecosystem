Electron JS est un framework basé sur Node.js et Chromium qui nous permet de construire des applications de bureau en utilisant des technologies web standard comme HTML, CSS et, bien sûr, JavaScript. Qu'est-ce que Electron ?

Electron.js est un cadre d'exécution qui permet à l'utilisateur de créer des applications de bureau avec HTML5, CSS et JavaScript. des applications de bureau avec HTML5, CSS et JavaScript. Il s'agit d'un projet open source lancé par Cheng Zhao, un ingénieur de GitHub.

Avantages Electron

Même les développeurs qui ne sont pas spécialisés dans le développement web frontal ont de nombreuses bonnes raisons d'adopter Electron, notamment:

  • sécurité
  • grande communauté de développeurs et d'utilisateurs
  • support multi-plateforme

Le développement Electron est le plus judicieux lorsqu'il s'agit de créer des applications de bureau multiplateformes qui n'ont pas besoin d'adhérer à des exigences strictes en matière d'utilisation de la mémoire. qui mettent l'accent sur la conception de l'UX et de l'UI.
