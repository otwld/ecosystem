Marketplace for audiovisual freelancers

Mission background

Onstage is a young startup that operates in the audiovisual sector. They contacted me with a web application project. The main goal of this platform is to put candidates in touch with missions they could carry out. My job was to transform with another freelancer this web application into their best weapon.

► The challenge was to provide an application with the core-features of a specialized talent platform for the audiovisual world.

Roles, actions taken and objectives achieved

  • Definition of the data architecture.
  • Integration & Features of all pages in Angular 11
  • Creation of the backend in NodeJS with NestJS
    • Access administration, news, management of candidate and advertiser profiles, mailings, quotation creation system with validation, interactive chat for mission discussion
    • Secure transactions under OAuth2
  • Implementation of a continuous integration and deployment environment under Github

Technical, business and functional environment

Technical: Angular NodeJS NestJS SCSS AWS TDD Kubernetes Docker Terraform Méthodologie Agile DevOps Jira

Functional: Project management, progress monitoring and planning, Facilitating and conducting meetings, Business process automation and workflow monitoring, Analysis of needs
