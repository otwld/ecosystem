Place de marché pour les indépendants de l'audiovisuel

Contexte mission

Onstage est une jeune startup qui opère dans le secteur de l'audiovisuel. Ils m'ont contacté pour un projet d'application web. L'objectif principal de cette plateforme est de mettre en relation des candidats avec des des missions qu'ils pourraient réaliser. Mon travail consistait à transformer avec un autre freelance cette application web application web en leur meilleure arme.

► Le défi consistait à fournir une application avec les caractéristiques principales d'une plateforme spécialisée dans les talents pour le monde de l'audiovisuel.

Rôles, actions menées et objectifs atteints

  • Définition de l'architecture des données.
  • Intégration et fonctionnalités de toutes les pages dans Angular 11
  • Création du backend en NodeJS avec NestJS
    • Administration des accès, actualités, gestion des profils des candidats et des annonceurs, mailings, système de création de devis avec validation, chat interactif pour la mission discussion
    • Transactions sécurisées sous OAuth2
  • Mise en place d'un environnement d'intégration et de déploiement continu sous Github

Environnement technique, commercial et fonctionnel

Technical: Angular NodeJS NestJS SCSS AWS TDD Kubernetes Docker Terraform Méthodologie Agile DevOps Jira
