Ateme

Contexte mission

En Aout 2021, j'ai rejoint l'entreprise Ateme pour renfort d'équipe Vue & NodeJs.

En travaillant conjointement avec l'équipe produit et l'équipe de design, j'ai pu concevoir un ensemble de fonctionnalités sur les différents applicatifs de l'entreprise.

► Le challenge était faire évoluer une application très large tout en maintenant un niveau de qualité très haut puisque le produit est utilisé pour de la retransmission en direct

Rôles, actions menées et objectifs atteints

  • Architecture & intégration de fonctionnalités dans une API NodeJs (NestJs) REST
  • Architecture & intégration de fonctionnalités dans une API NodeJs (NestJs) GraphQl en microservice
  • Conception et développement de fonctionnalités sur plusieurs apps VueJs
  • Gestion des déploiements des applicatifs sur kubernetes
  • Gestion d'équipe et de tâches avec le Framework Scrum
    • Gestion du tableau de tâche & des avancés de l'équipe
    • Participation aux rituels
    • Formations techniques pour les autres équipes
  • Collaboration avec les QAs pour la mise en place de tests automatisés
  • Documentation des features sur confluence pour les QAs
  • Démonstration d'avancement devant l'entreprise complète
  • Réunion hebdomadaire/quotidienne au réunion produit / ciblage du besoin.
